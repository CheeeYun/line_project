# Linebot 

自動回覆+linePay金流+精準客戶推播

## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:10000`:

```bash
# pnpm
pnpm run dev
```

## Production

Build the application for production:

```bash
nuxt start
```
